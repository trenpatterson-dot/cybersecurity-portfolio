# Module 1 – Network Traffic Analysis
### CYBR 445 | Bellevue University

---

## 🎯 Lab Objectives

- Capture and analyze network traffic using Wireshark
- Identify protocols, conversations, and anomalies in packet captures
- Understand how network monitoring fits into a SOC workflow

---

## 🧠 Key Concepts Learned

- **Packet capture** using Wireshark and tcpdump
- **Protocol analysis** — TCP, UDP, HTTP, DNS, ICMP
- **Display filters** to isolate specific traffic
- **Follow TCP Stream** to reconstruct conversations
- Identifying **suspicious traffic patterns** and anomalies

---

## 🔑 Key Takeaways

1. Network visibility is foundational to incident detection
2. Wireshark filters dramatically speed up analysis
3. Unencrypted protocols expose sensitive data in transit
4. Baseline traffic knowledge is essential for anomaly detection

---

## 🛠️ Tools Used

- Wireshark
- tcpdump
- Linux CLI
