PORTFOLIO AGENT - QUICK COMMANDS

1. Run one project
python agent.py --project phishing-analysis

2. Run one project and show output file paths
python agent.py --project phishing-analysis --open

3. Preview one project in public-safe mode without writing files
python agent.py --project phishing-analysis --public --dry-run --open

4. Pick a project from a list
python agent.py --pick --open

5. Run all valid portfolio projects
python agent.py

6. Run only projects changed in the last 7 days
python agent.py --recent-days 7

7. Preview everything without writing files
python agent.py --dry-run

8. Preview one project without writing files
python agent.py --project phishing-analysis --dry-run

9. Run one project in public mode
python agent.py --project phishing-analysis --public

WARNING
Do not use:
python agent.py --project phishing-analysis --project cyber-deception

Repeated --project values are not supported yet.
