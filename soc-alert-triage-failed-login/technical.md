## Overview
The Soc Alert Triage Failed Login is a detection engineering exercise documented under Wazuh. The lab aims to investigate and analyze failed login attempts, providing insights into potential security threats and defensive measures.

## Tools & Environment
The primary tools used in this lab are Wireshark for network traffic analysis and Wazuh for security monitoring and alerting. The environment is Windows-based, with the specific path being `C:\Users\trenp\cybersecurity-portfolio\security-plus-projects\soc-alert-triage-failed-login`.

## Security Significance
This lab serves as a practical exercise for understanding and responding to failed login attempts, which can be indicative of potential brute-force attacks or credential stuffing attempts. By analyzing these events, cybersecurity professionals can improve their organization's defense against such threats.
