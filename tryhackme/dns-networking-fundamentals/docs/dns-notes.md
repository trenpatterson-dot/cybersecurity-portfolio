# DNS Notes

## What is DNS?
DNS (Domain Name System) translates domain names into IP addresses.

## Key Records
- A → IPv4
- AAAA → IPv6
- MX → Email routing
- CNAME → Alias

## Important Terms
- TTL → Cache duration
- Recursive Server → Finds answers
- Authoritative Server → Holds real records

## Key Concept
DNS = Name → IP translator
