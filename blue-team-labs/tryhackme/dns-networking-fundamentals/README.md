## 🧠 Key Concepts Learned

### 🔐 Firewall
- Controls inbound and outbound traffic using rules
- Operates primarily at OSI Layers 3 & 4
- Can allow or deny traffic based on IP, port, and protocol

---

### 🌐 DNS (Domain Name System)
- Translates domain names into IP addresses
- Key record types:
  - A → IPv4 address
  - AAAA → IPv6 address
  - MX → Mail routing
  - CNAME → Alias

---

### 🔁 DNS Process
1. User enters domain name
2. Recursive resolver searches for the IP
3. Query passes through root and TLD servers
4. Authoritative server provides final answer

---

### 🔒 VPN (Virtual Private Network)
- Encrypts network traffic
- Hides IP address
- Creates secure communication tunnels

---

### 🧱 Networking Devices
- Router → connects networks
- Switch → directs traffic to specific devices
- Hub → broadcasts to all devices
- Access Point → provides wireless connectivity
