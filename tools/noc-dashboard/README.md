# NOC Support Dashboard

## Project Overview
AI-powered NOC Technical Support Dashboard built with HTML, CSS, JavaScript, and Python.

## What I Built
- Full ticket queue system with priority levels (P1-P4) and status tracking
- Network device status board with real-time indicators
- Client records and troubleshooting runbooks
- AI Assist feature powered by Claude API via a local Python proxy server

## Tools Used
- HTML / CSS / JavaScript (vanilla)
- Python (http.server + custom proxy)
- Anthropic Claude API (claude-haiku)

## Skills Demonstrated
- Full-stack web development
- API integration and proxy architecture
- CORS troubleshooting and local server configuration
- UI/UX design for operational dashboards
